After installing the module, assign users to the appropriate approval groups
under **Settings → Users & Companies → Users → (select user) → Other**:

- **Quotation Approval / Team Leader** — can approve quotations that
  require Team Leader sign-off.
- **Quotation Approval / Sales Manager** — can approve at the Sales Manager
  level in the full 3-level flow.
- **Quotation Approval / Finance Manager** — provides the final approval
  when selling below cost.

Products must have a **Cost** (`standard_price`) set for the approval level
to be computed correctly.
